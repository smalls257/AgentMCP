# AgentMCP (Agent Mission Control)

A high-performance native desktop app for managing, monitoring, and steering AI agents across workspaces.

## Architecture
- **Frontend**: Svelte (compiled to vanilla JS)
- **Backend**: Tauri 2 (Rust)
- **Providers**: GitHub Copilot CLI SDK, Claude Code, Gemini CLI, OpenRouter.
